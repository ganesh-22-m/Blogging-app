📝 Blogging Application (Full Stack)

🚀 Overview
A full-stack Blogging Application built using modern web technologies. This platform allows users to create, manage, 
and interact with blog posts with secure authentication and a responsive user interface.

🧱 Tech Stack
🔹 Backend
Java
Spring Boot
Spring Data JPA
Spring Security (JWT Authentication)

🔹 Frontend
React.js
HTML5, CSS3, JavaScript
Axios

🔹 Database
MySQL

🔹 Tools & Platforms
Git & GitHub

Postman
IntelliJ IDEA / VS Code

✨ Features
🔐 User Authentication (Register & Login with JWT)
📝 Create, Update, Delete Blog Posts
📖 View All Blogs
💬 Comment on Blogs
❤️ Like / Unlike Posts
🔍 Search Functionality
📄 Pagination Support
🛡️ Role-Based Access Control (Admin/User)
📱 Responsive UI Design
